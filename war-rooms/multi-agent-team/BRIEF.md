# 多智能体团队系统项目简报

## 项目目标
构建一个可重用的多智能体协作系统，支持：
- 总指挥: Gemini 3.0 (决策、调度)
- 图像生成: Nano Banana Pro (文生图、图生图)  
- 代码任务: Claude Code (编程实现)
- 执行协调: 小爪 (Qwen, 本地执行、文件管理、API 调用)

## 预期功能
- 深度分析任务的多智能体协作
- 图像生成需求的自动路由
- 代码编写任务的智能分配
- 工作流协调与状态跟踪

## 当前状态
- 系统架构待设计
- API Keys 待采购
- 配置框架待搭建
- 通信协议待定义

## 约束条件
- 所有 API 配置需安全存储
- Agent 间通信需高效可靠
- 系统应易于扩展和维护
- 支持离线模式（仅启用已有服务）
