## 📚 巴菲特投资核心理念

### 1. 价值投资四大原则

**原则一：把股票当作企业的一部分**
- 买股票 = 买企业所有权
- 关注企业本身，而非股价波动
- "如果你不愿意持有一只股票 10 年，就不要持有它 10 分钟"

**原则二：安全边际**
- 用 4 毛钱买价值 1 块钱的东西
- 内在价值 > 市场价格
- 留足容错空间

**原则三：市场先生**
- 市场是仆人，不是向导
- 利用市场情绪，而非被情绪左右
- "别人贪婪时恐惧，别人恐惧时贪婪"

**原则四：能力圈**
- 只投资自己理解的企业
- 坚守认知边界
- 宁可错过，不要错投

## 🏰 护城河理论（经济护城河）

### 五种护城河类型

| 护城河类型 | 说明 | 典型案例 |
|-----------|------|---------|
| **无形资产** | 品牌、专利、特许经营权 | 可口可乐、贵州茅台 |
| **转换成本** | 客户更换供应商成本高 | 微软 Office、Adobe |
| **网络效应** | 用户越多价值越大 | 微信、Visa、Mastercard |
| **成本优势** | 规模效应带来的低成本 | 沃尔玛、比亚迪 |
| **有效规模** | 市场容量有限，新进入者无利可图 | 机场、港口 |

### 护城河评估标准

✅ **宽护城河**：预计 20 年以上竞争优势  
✅ **窄护城河**：预计 10-20 年竞争优势  
⚠️ **无护城河**：竞争优势不明或不足 10 年

## 📊 巴菲特选股财务指标

### 核心指标要求

| 指标 | 巴菲特标准 | 说明 |
|------|-----------|------|
| **ROE（净资产收益率）** | > 15% 连续 5-10 年 | 盈利能力核心指标 |
| **毛利率** | > 40% | 产品竞争力体现 |
| **净利率** | > 15% | 最终赚钱能力 |
| **负债率** | < 50% | 财务安全 |
| **自由现金流** | 持续为正且增长 | 真金白银 |
| **股息率** | 稳定且增长 | 股东回报 |

### 自由现金流公式

```
自由现金流 = 经营现金流 - 资本支出
```

**巴菲特说**："现金流是王，利润是观点"

## 💰 估值方法

### 1. 市盈率（PE）法

```
合理 PE = 1 / 无风险收益率
```

- 当前无风险收益率（10 年期国债）≈ 2.5%
- 合理 PE ≈ 40 倍
- **但巴菲特会打折扣**：安全边际 → 20-25 倍买入

### 2. DCF 现金流折现法

```
内在价值 = Σ (未来自由现金流 / 折现率)
```

**简化版**：
- 预估未来 10 年自由现金流
- 折现率取 8-10%（巴菲特用 10 年期国债利率 + 风险溢价）
- 加上终值
- 打 6-7 折作为买入价（安全边际）

### 3. 格雷厄姆公式

```
内在价值 = EPS × (8.5 + 2g)
```

- EPS: 每股收益
- g: 预期年增长率（%）
- 8.5: 零增长公司的合理 PE

## 🎯 巴菲特式选股流程

### Step 1: 初筛（定量）

```
1. ROE > 15% 连续 5 年
2. 负债率 < 50%
3. 毛利率 > 40%
4. 自由现金流持续为正
5. 股息率 > 2%（可选）
```

### Step 2: 护城河分析（定性）

```
1. 品牌优势？
2. 转换成本高？
3. 网络效应？
4. 成本领先？
5. 特许经营？
```

### Step 3: 管理层评估

```
1. 诚信记录
2. 资本配置能力
3. 股东利益导向
4. 长期思维
```

### Step 4: 估值

```
1. 计算内在价值（DCF）
2. 对比当前市值
3. 等待安全边际（至少 30% 折扣）
```

### Step 5: 买入并持有

```
1. 分批建仓
2. 长期持有（5-10 年+）
3. 定期复查基本面
4. 除非基本面恶化，否则不卖出
```

## 🚫 巴菲特不投什么

1. **看不懂的科技股**（早期不投亚马逊、谷歌）
2. **周期股**（航空、航运、大宗商品）
3. **初创企业**（没有稳定现金流）
4. **高负债企业**（财务风险大）
5. **频繁并购的企业**（资本配置不当）
6. **财务造假嫌疑**（诚信问题一票否决）
