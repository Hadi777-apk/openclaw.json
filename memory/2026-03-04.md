## 🎯 核心知识点

### 巴菲特投资四大原则
1. **企业思维** - 买股票=买企业所有权
2. **安全边际** - 用 4 毛买 1 块的东西
3. **市场先生** - 利用情绪，不被情绪左右
4. **能力圈** - 只投理解的企业

### 护城河五种类型
- 无形资产（品牌、专利、特许）
- 转换成本
- 网络效应
- 成本优势
- 有效规模

### 选股财务标准
| 指标 | 标准 |
|------|------|
| ROE | >15% 连续 5-10 年 |
| 毛利率 | >40% |
| 净利率 | >15% |
| 负债率 | <50% |
| 自由现金流 | 持续为正且增长 |
| 股息率 | >2%（可选）|

### 估值方法
1. PE 法 - 合理 PE 20-25 倍买入
2. DCF 现金流折现
3. 格雷厄姆公式
