## 🎯 重要决定

### 2026-02-25 - 记忆系统策略

**决定**：使用 OpenClaw 内置文件基础记忆系统，暂不安装外部记忆插件

**理由**：
1. 内置系统已满足核心需求（跨会话持久化）
2. 文件基础 = 安全、可控、可查看
3. 避免额外依赖和复杂度
4. 外部项目（如 MemOS）可能重复造轮子

**关于 openclaw-foundry（自我进化插件）**：
- 概念很酷，但有安全风险
- 自动编写并运行代码需要极高信任度
- 暂不安装，除非有明确需求且经过严格评估
