# 微信公众号自动发布技能

## 概述

这个技能可以实现微信公众号的自动化运营，包括：
- 自动发布文章
- 定时推送
- 数据监控
- 自动回复（需要企业号/服务号）

## 微信公众号类型与API支持

| 类型 | API支持 | 自动发布 |
|------|---------|----------|
| 订阅号（个人） | ❌ 无 | 需手动登录 |
| 订阅号（企业） | ⚠️ 有限 | 需申请 |
| 服务号 | ✅ 有 | 可通过API |
| 企业微信 | ✅ 有 | 完善API |

## 实现方案

### 方案1：手动扫码 + 定时发布（订阅号）

适用于个人订阅号：
1. 扫码登录公众号后台（需要每7天一次）
2. 手动发布第一篇文章
3. 使用工具复制多开
4. 定时任务触发

### 方案2：WeChat Pub API（服务号）

需要条件：
- 已认证的服务号
- 已认证的企业资质
- 有开发能力

API接口示例：
```python
# 微信公众号草稿箱API
POST https://api.weixin.qq.com/cgi-bin/draft/add?access_token=TOKEN

# 发布文章
POST https://api.weixin.qq.com/cgi-bin/freepublish/submit?access_token=TOKEN
```

### 方案3：第三方工具

推荐工具：
- 微盛·企微助手
- 有赞
- 蝉妈妈
- 新榜

## 当前推荐方案

由于你的订阅号是个人类型，推荐方案：

1. **使用 Chrome DevTools MCP 自动发布**
   - 扫码登录一次
   - 自动化填写表单发布

2. **每7天重新扫码登录**

## 使用方法

### 步骤1：扫码登录

```bash
# 打开公众号登录页
mcporter call chrome-devtools.navigate_page type=url url=https://mp.weixin.qq.com/
```

### 步骤2：发布文章

需要的元素：
- 标题输入框
- 正文编辑区
- 封面图上传
- 群发按钮

### 步骤3：定时任务

使用 cron 定时执行发布脚本

## 注意事项

1. **个人订阅号没有API**
2. **每次发布需要素材管理**
3. **注意规避诱导分享规则**
4. **二维码7天有效**
